> Why pay for a masterclass when you can swap skills with your neighbor? The Local Skill Swap Hub connects community members who want to teach what they know and learn what they don't. 

## ✨ Key Features

This application operates entirely in the browser without a backend, acting as a rich, robust prototype leveraging `localStorage` and complex Document Object Model manipulation:

- **Intelligent Mutual Matching**: Automatically identifies users who offer exactly what you want, *and* want exactly what you offer.
- **Async Notification System**: Simulated active inbox that pops alert menus organically with realistic latency delays. 
- **Persisted State & Bookmarks**: Add listings, save favorites, and securely filter your active feed all locally.
- **Dynamic Avatars & Online Tracking**: Automatic realistic API avatar rendering and conditional online-status CSS tokens.
- **Interactive Review System**: Live star-rating calculators capable of adjusting user data properties on-the-fly.
- **New User Onboarding**: Dynamic spotlight overlay system mapping UI components iteratively for first-time visitors.

## 🚀 How to Run Locally

Because this project relies completely on core web technologies:

1. Clone or download this repository.
2. Navigate to the project folder.
3. Simply launch **`index.html`** in any modern web browser.
*(Ensure JavaScript is enabled to interact with the database elements).*

## 🌐 Deploying Live

This repository is inherently deploy-ready for any static asset platform like **Netlify, Vercel, or GitHub Pages**.

**To deploy instantly via GitHub Pages**:
1. Commit these files to a public repository.
2. Navigate to Repo **Settings** > **Pages**.
3. Under Build and Deployment, target your `main` branch.
4. Save. Your live Local Skill Swap Hub is born!
